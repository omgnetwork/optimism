# Setting up Minikube

N.B. What follows doesn't work while storage classes come from GCP.
TODO: fix.

## Start Minikube

Execute:

```bash
minikube start --mount=true
```

## Ending Minikube

To stop the minikube VM:

```bash
minikube stop
```

To delete the minikube VM:

```bash
minikube delete
```
